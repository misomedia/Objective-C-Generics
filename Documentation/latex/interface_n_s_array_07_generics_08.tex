\subsection{Detailed Description}
A category on N\-S\-Array to provide Haskelly functional programming functionality with respect to (parametrically polymorphic or very lucky) arrays. Ad hoc polymorphism will behave in an ad hoc manner.

These methods do not directly sort a list using a comparator to project. Instead, they construct a dictionary whose keys are the image of the list under a projection function and whose values are list of objects from the original list with the same projection. Then they sort the keys using a comparator O\-N T\-H\-E P\-R\-O\-J\-E\-C\-T\-I\-O\-N\-S (which can assume that the inputs are not equal). Then they map to get a list of lists. Then they flatten the list of lists to get the result.
